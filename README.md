# 🌦️ WeatherReport

A weather forecasting web application built using **ASP.NET Web Forms**, **C#**, and the **OpenWeatherMap API**. WeatherReport provides real-time weather information, location-based forecasts, and city bookmarking through an intuitive dashboard.

---

## 📌 Overview

WeatherReport is a full-stack web application that retrieves live weather data using the OpenWeatherMap API and presents it through an interactive web interface. Users can search for cities, view current weather conditions, bookmark their favorite locations, and explore weather statistics such as the hottest and coldest cities. The application integrates MySQL for user management and bookmarked locations while providing a responsive and user-friendly experience.

---

## ✨ Features

- 🌍 Real-time Weather Information
- 🔍 City Search
- 📍 Location-based Weather
- ⭐ Bookmark Favorite Cities
- 🌡️ Hottest & Coldest Cities Dashboard
- 👤 User Authentication
- 🗄️ MySQL Database Integration
- 📊 Interactive Weather Dashboard
